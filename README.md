# Bank Management System

A console-based Bank Management System built in C++ to simulate core banking operations using Object-Oriented Programming (OOP), Standard Template Library (STL), and File Handling.

## Features

- Create a new bank account
- Search account by account number
- Deposit money
- Withdraw money
- Check account balance
- View transaction history
- Store account information using file handling
- Load saved account data on program startup
- Input validation for common user errors

## Tech Stack

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
- File Handling
